ТЗ: Task Manager (упрощенная версия)
Цель: Изучить React за 1-2 дня на практическом проекте
Технический стек

React 19 + TypeScript + Vite
React Router (базовый роутинг)
Zustand (1-2 store)
React Hook Form + Zod
Shadcn/ui + Tailwind
JSON Server для fake API

Функционал (минимум)
День 1: Основы

Auth (mock):

Простая форма логина
Zustand store для user
Protected route


Tasks CRUD:

Список задач
Добавить/редактировать/удалить
Поля: title, description, status, priority


Базовый UI:

Header с logout
Sidebar навигация
Task list компонент



День 2: Продвинутое

Фильтрация:

Поиск по названию
Фильтр по статусу
Debounced search


State management:

React Query для tasks API
Оптимистичные обновления
Error handling


Полировка:

Loading states
Toast уведомления
Dark/light theme



Структура (упрощенная)
src/
├── components/ui/    # 3-4 shadcn компонента
├── features/
│   ├── auth/        # Login + store
│   └── tasks/       # CRUD + список
├── hooks/           # 2-3 custom hooks
├── lib/             # API client
├── stores/          # authStore + settingsStore
└── pages/           # 3 страницы
Маршруты (минимум)
/login     # Форма входа
/tasks     # Список задач (главная)
/tasks/:id # Редактирование задачи
API (JSON Server)
json{
  "tasks": [
    {
      "id": 1,
      "title": "Learn React",
      "description": "Study hooks and state management",
      "status": "todo|in-progress|done",
      "priority": "low|medium|high",
      "createdAt": "2025-01-27"
    }
  ],
  "user": {
    "id": 1,
    "email": "user@example.com",
    "name": "John Doe"
  }
}
